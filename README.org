* Ruby Khipu

A Ruby investigation and implementation of the Khipu data structure described
by [[https://ccsenet.org/journal/index.php/cis/article/view/0/52950][this paper.]]

See the literate program [[./khipu.org]].

** Example

The ~test.rb~ file isn't currently real tests, but when run intends to exercise
and output the serialized versions of the data structure.

#+begin_example
An empty cord =>
|||
A red cord with knots =>
|(255, 0, 0, 255)||12345
A red cord facing up with knots =>
|(255, 0, 0, 255)|1|12345

1|||
  |||
  
A group with a knotted top cord =>
1|||
  |||12345
  
A group with top cord and red cord =>
1|||

  |(255, 0, 0, 255)|1|12345

  |||12345
