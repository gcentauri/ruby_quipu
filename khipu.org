* Ruby Khipu

A Ruby investigation and implementation of the following paper:

https://ccsenet.org/journal/index.php/cis/article/view/0/52950

We'll start by implementing my understanding of khipu as an organized system of
integer data. The paper keeps saying a khipu id "heirarchical, partially
unsorted, grouped and summed". Its a tree structure that allows arbitrary O(1)
inserts (heirarchical and unsorted). Elements of the tree can be grouped, and
groups have a built in summation via the "top-cord" feature.

** A Cord of Knots is an Integer

Each cord encodes a number in base 10, with each knot being a digit.
Naively, we could implement this as an array of integers from 0-9. In the most
basic interpretation, we read the knots as an integer.

#+begin_src ruby
  cord = Class.new do
    attr_reader :knots

    def initialize
      @knots = []
    end

    def <<(val) = @knots << val
    def to_s = @knots.join
    def to_i = to_s.to_i
  end

  a_cord = cord.new
  a_cord << 1
  a_cord << 2
  a_cord << 3

  [a_cord.knots, a_cord.to_i]
#+end_src

#+RESULTS:
| (1 2 3) | 123 |

Adding a knot for 1 in the hundreds place, 2 in the tens, and 3 in the units
we get the integer 123.

** Strand base class

The paper's interface proposes a base class that is an iterable. I debate about
simply inheriting from ~Array~ to build up the Khipu from Groups of Cords. But the
standard way of making an iterable in Ruby is to include ~Enumerable~ and define
~each~. May as well use an array as the object member that holds ~knots~.

#+begin_src ruby :tangle yes
  class Strand
    include Enumerable

    attr_accessor :knots

    def initialize
      @knots = []
      @group = false
      @colour = nil
      @direction = false
    end

    def each(&block) = knots.each(&block)
    def <<(val) = knots << val
    def length = count
    def insert(i, x) = knots[i] = x
    def append(item) = knots.append(item)
    def concat(strand) = strand.each { |item| append(item) }
    def clear = knots.clear
    def copy = Marshal.load(Marshal.dump(self))

    # TODO
    def decrypt(key) = nil
    def encrypt = nil

    def to_s
      "#{group ? 1 : ''}|#{colour}|#{direction ? 1 : ''}|#{knots.join}"
    end

    private

    attr_reader   :group,
                  :colour,
                  :direction
  end
#+end_src

  class Cord < Strand
    def colour=(rgba_array)
      @colour = Colour.new(*rgba_array)
    end

    def direction=(bool)
      @direction = bool
    end

    def +(cord)
      new_value = to_i + cord.to_i
      @knots = new_value.to_s.chars.map(&:to_i)
      self
    end

    def to_i = knots.join.to_i

      def initialize(r,g,b,a)

      def green = @colour[1]
      def blue = @colour[2]

** Group

#+begin_src ruby :tangle yes
  class Group < Strand
    attr_reader :top_cord

    def initialize(has_top = true)
      super()
      @group = true
      @has_top = has_top
      @top_cord = Cord.new if has_top?
    end

    def has_top? = !!@has_top

    def add_to_top_cord(cord)
      if has_top?
        @top_cord += cord
      else
        @top_cord = Cord.new
        @has_top = true
        @top_cord += cord
      end
    end

    def append(cord, top_cord = true)
      @knots << cord

      add_to_top_cord(cord) if top_cord
    end

    def to_s = ["1|||", @top_cord, knots].join("\n  ")

    def to_i
      return top_cord.to_i if has_top?

      knots.map(&:to_i).sum
    end

    def clear
      knots.clear
      @top_cord = Cord.new
      self
    end

    def concat(cords, update_top_cord = true)
      cords.each { |cord|
        add_to_top_cord(cord) if update_top_cord && has_top?
        knots << cord
      }
    end

    def insert(i, cord, top_cord = true)
      knots[i] = cord

      add_to_top_cord(cord) if top_cord
    end
  end
#+end_src
